# Retold Beacon Host

> Generic container host for ultravisor-beacon CapabilityProvider classes published as npm packages

- Load one or more provider packages by name
- Serve them through a single beacon and Orator server
- Register with a target Ultravisor
- Container-first, with a developer-box `npx` path
